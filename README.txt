First, you need to open the command line directly in this folder.


Then follow these steps:


1.Run the command " npm install " and wait for the packages to download;


2.If you want to start the server, you must run the command: " npm run dev ";


3.If you want to create a ready-made assembly for uploading to the server, then run the command " npm run build ", 

after that, a finished assembly is formed in the " dist " folder for uploading to the server with the entry point " index.js ".
